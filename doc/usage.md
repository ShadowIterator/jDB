# 使用说明

## 启动服务器

```bash
cd deployed
java -jar jDBServer.jar
```

## 启动图形界面客户端

```bash
cd deployed
java -jar jDBGUI.jar
```

## GUI

![gui](D:\THU\2018-2019春\计算机系统软件2\project\jDB\usage\gui.PNG)

1. sql语句输入到左边的输入框
2. 点击`Execute`按钮
3. 在右边的输出框可以看到输出
4. 在下边栏可以看到运行时间

## Import

![import](D:\THU\2018-2019春\计算机系统软件2\project\jDB\usage\import.PNG)

1. 点击菜单栏的`file`
2. 选择`import sql script`选项卡
3. 在弹出的文件框中选择文件
4. 选择完成之后，语句自动执行，不需要点击按钮
5. 在右边的输出框看到输出
6. 在下边栏看到运行时间
